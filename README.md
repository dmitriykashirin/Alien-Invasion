# Alien Invasion

## Описание 
Alien Invasion - это аркадная игра, в которой космический корабль противостоит 
инопланетному флоту.
Корабль перемещается вдоль нижней части экрана влево и вправо. Корабль стреляет 
снарядами, которые уничтожают пришельцев по одному.
Флот пришельцев движется в сторону, как только флот касается края экрана, он 
смещается вниз и начинает двигаться в другую сторону. Так происходит пока флот
не достигнет нижнего края экрана, не столкнётся с кораблём или не будет 
уничтожен. 
У игрока имеется 3 жизни. При столкновении пришельца и корабля, или если флот 
коснулся нижнего края экрана, или если флот уничтожен, то все игровые элементы
занимают стартовые позиции, а игрок теряет 1 жизнь. С течением игры, скорость 
всех игровых элементов возрастает.
По истечении всех жизней игра заканчивается и игроку предлагается начать её 
сначала.

### Технологии
* **Python 3.13**
* **Pygame** для графики
* **ООП** для структурирования кода

### Основные функции
* Управление космическим кораблем
* Система стрельбы
* Волны пришельцев

### Технические детали
* **Архитектура**: модульное проектирование
* **Коллизии**: реализация системы столкновений

### Установка и запуск
* Установка зависимостей: pip install -r requirements.txt
* Запуск игры происходит через главный файл: alien_invasion.py
